Export/Import

Export

Download a ZIP file containing all your smart playlist configurations. This can be used as a backup or to transfer playlists to another Jellyfin instance.

Import

Select a ZIP file exported from SmartPlaylist plugin. Playlists with duplicate IDs will be skipped to avoid conflicts. After importing, you need to refresh all playlists to create them in Jellyfin.
